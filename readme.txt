# weather_bot

Телеграм бот отвечающий на сообщение с датой, показом прогноза погоды в городе Ярославль
В качестве источника, берёт данные со страницы яндекс погоды и извлекает их с помощью Beautiful Soup

Для установки:
Создать виртуальное окружение и установить зависимости из файла requirements.txt
Добавить токен вашего бота в переменную в файле config.py
Запустить файл bot.py
